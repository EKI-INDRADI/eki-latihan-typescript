##  EKI TYPESCIPT BASIC 
```sh
npm install -g typescript
```

```sh
tsc --version
```

##  typescript akan di compile menjadi javacript menggunakan perintah tsc

##  compile TS to JS
```sh
tsc 1_app.ts
```

##  RUN JS
```sh
node 1_app.js
```

##  run bersamaan : 
```sh
tsc 1_app.ts & node 1_app.js 
```

##  atau jika ingin lebih mudah
```sh
npm install -g ts-node
```

```sh
ts-node 1_app.ts
```
